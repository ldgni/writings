# Writings

> My writings

## Running Locally

1. Clone the repository

```sh
git clone https://github.com/ldgni/writings.git
```

2. Navigate inside the repository

```sh
cd writings
```

3. Install dependencies

```sh
npm install
```

4. Start the development server

```sh
npm run dev
```
